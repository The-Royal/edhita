Edhita
======

Fully open source text editor for iOS written in Swift.  
http://edhita.bornneet.com/

![](/screenshot.png)

## What Edhita means?

`Edhita (Romaji)` == `エディタ (Katakana)` == `Editor (English)`

## Development

```
$ cp AppSecret.swift Edhita/Models/
$ pod install
$ open Edhita.xcworkspace
```
